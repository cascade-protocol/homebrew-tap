# Cascade Homebrew Tap

Official Homebrew tap for [Cascade](https://cascade.fyi) CLI tools.

## Installation

```bash
brew tap cascade-protocol/tap
brew install cascade
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `cascade` | Connect your local MCP server to Cascade Market |

## Updating

```bash
brew update
brew upgrade cascade
```

## Uninstalling

```bash
brew uninstall cascade
brew untap cascade-protocol/tap
```
